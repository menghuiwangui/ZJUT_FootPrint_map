# 🗺️ Footprint Map - 个人足迹地图

一个前后端分离的足迹记录与分享平台，支持地图打点、游记管理、旅行数据统计与年度足迹报告。

## ✨ 功能特性

- 📍 地图标记去过的地方，支持搜索与拖拽定位
- 📝 发布游记，上传照片，记录旅行故事
- 📊 数据可视化：出行频率、热力图、足迹统计
- 📅 年度足迹报告生成与分享
- 🔐 JWT 身份认证

## 🛠️ 技术栈

### Frontend
- Vue3 + Vite + Pinia
- 高德地图 SDK
- ECharts

### Backend
- Spring Boot 3 / Node.js (NestJS)
- MySQL 8.0 + Redis
- JWT + Swagger

## 📁 项目结构

footprint-map/
├── frontend/    # 前端应用
├── backend/     # 后端服务
└── docs/        # 文档与资源
